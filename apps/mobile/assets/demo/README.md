# Demo step images (camera overlay)

Put your **8 demo images** here. They are shown in the camera "demo" overlay before the user taps "Passer".

**Expected filenames:**
- `step_1.png`
- `step_2.png`
- …
- `step_8.png`

You can use `.jpg` instead of `.png`; if you do, either rename to `.png` or change the paths in `camera_screen.dart` (`_demoStepAssets`).
